# Global Pixel Wall

A real-time, collaborative art project where users can claim and color pixels on a shared 20x20 grid. This project demonstrates a full-stack architecture using a Python backend and a modern JavaScript frontend.

## Features
- **Persistent Grid:** Pixel data is saved in a JSON database on the server, so your art stays even after a refresh.
- **Interactive UI:** A smooth, dark-mode interface with hover effects and real-time local updates.
- **Full-Stack Integration:** Uses a Flask API to bridge the gap between the frontend and the data storage.

## Tech Stack
- **Backend:** Python 3.x, Flask
- **Frontend:** HTML5, CSS3 (Grid & Variables), Vanilla JavaScript
- **Data:** JSON-based persistent storage

## Installation & Setup
1. **Clone the repository:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/global-pixel-wall.git](https://github.com/YOUR_USERNAME/global-pixel-wall.git)
   cd global-pixel-wall
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   python app.py
   ```
